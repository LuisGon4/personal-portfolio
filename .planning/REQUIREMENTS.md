# Requirements: Personal Portfolio Website — v1.1.0

**Defined:** 2026-03-05
**Milestone:** v1.1.0 — Visual Rework

## v1.1.0 Requirements

### Navigation

- [ ] **NAV-01**: Nav bar uses black background with white text on all breakpoints
- [ ] **NAV-02**: Mobile hamburger menu matches black theme; hover inverts (white bg / black text)

### Skills Section

- [ ] **SKILL-01**: A "Technologies" section exists between About and Projects
- [ ] **SKILL-02**: Skills are grouped by category (Languages, Frameworks, Tools) with category labels
- [ ] **SKILL-03**: Skill tag pills match the existing ProjectCard tag style (mono, border, indigo hover)
- [ ] **SKILL-04**: Nav includes a "Skills" link that smooth-scrolls to the section

### Hero Decorations

- [ ] **SHAPE-01**: Geometric shapes in the hero section do not overlap or obscure text content
- [ ] **SHAPE-02**: Additional geometric shapes of varied type (lines, rectangles, dots, etc.) added to hero
- [ ] **SHAPE-03**: All decorative shapes are aria-hidden and do not interfere with pointer events

## Out of Scope

| Feature | Reason |
|---------|--------|
| Dark mode toggle | Scope creep — not requested |
| Animated shapes | Conflicts with no-gratuitous-animation philosophy |
| Skill proficiency levels | Adds complexity; keep it scannable |
