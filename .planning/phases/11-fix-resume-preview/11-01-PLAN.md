# Plan 11-01: Resume Preview Toggle

## Objective

Add a toggle button to `src/sections/Resume.jsx` so the PDF iframe is hidden on initial load. A "Preview Resume" button reveals it; a "Hide Preview" button hides it again.

## Context

- **Only file changed:** `src/sections/Resume.jsx`
- **Current state:** Iframe always renders on load
- **Target:** Iframe hidden by default, shown only when user clicks toggle

## Tasks

<task id="11-01-01" type="code">
Add useState import and toggle state to Resume.jsx

- Add `import { useState } from 'react';` at top of file
- Add `const [showPreview, setShowPreview] = useState(false);` inside component
</task>

<task id="11-01-02" type="code">
Replace download link with flex row containing Download + Preview toggle button

Replace the standalone `<a>` download link with a flex container:
```jsx
<div className="flex items-center gap-6">
  <a
    href="/resume.pdf"
    download
    className="inline-block underline underline-offset-4 hover:text-black/60 transition-colors"
  >
    Download Resume
  </a>
  <button
    onClick={() => setShowPreview(p => !p)}
    className="border border-black px-4 py-1.5 text-sm hover:bg-black hover:text-white transition-colors"
  >
    {showPreview ? 'Hide Preview' : 'Preview Resume'}
  </button>
</div>
```
</task>

<task id="11-01-03" type="code">
Conditionally render iframe based on showPreview state

Wrap the `<div className="mt-8 w-full">` block containing the iframe in `{showPreview && (...)}` so it only renders when toggle is open.
</task>

## Verification Criteria

- On page load: iframe is not visible, button reads "Preview Resume"
- After clicking: iframe appears, button reads "Hide Preview"
- After clicking again: iframe hides, button resets to "Preview Resume"
- Download Resume link still works
- Button uses outlined style (black border, transparent bg, inverts on hover)
- No animation — instant show/hide

## must_haves

- [ ] `showPreview` state defaults to `false`
- [ ] Button label toggles between "Preview Resume" and "Hide Preview"
- [ ] Iframe not rendered when `showPreview` is false
- [ ] Download link and toggle button are in same row (flex)
- [ ] Button style: `border border-black` outlined pattern
