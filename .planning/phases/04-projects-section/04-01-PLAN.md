Write src/components/ProjectCard.jsx. This is a pure presentational component — no state, no side effects, no imports beyond React (which is auto-imported in React 19 / Vite).

Exact implementation:

```jsx
export default function ProjectCard({ title, description, tags, githubUrl, liveUrl }) {
  return (
    <article className="border border-black flex flex-col gap-4 p-6">
      <h3 className="text-xl font-semibold tracking-tight">{title}</h3>
      <p className="text-sm leading-relaxed text-black/70 flex-1">{description}</p>
      <ul className="flex flex-wrap gap-2">
        {tags.map((tag) => (
          <li
            key={tag}
            className="font-mono text-xs px-2 py-1 border border-black/30 tracking-wide"
          >
            {tag}
          </li>
        ))}
      </ul>
      <div className="flex gap-4 text-sm">
        <a
          href={githubUrl}
          target="_blank"
          rel="noopener noreferrer"
          className="underline underline-offset-4 tracking-wide hover:text-black/60"
        >
          GitHub
        </a>
        {liveUrl && (
          <a
            href={liveUrl}
            target="_blank"
            rel="noopener noreferrer"
            className="underline underline-offset-4 tracking-wide hover:text-black/60"
          >
            Live Demo
          </a>
        )}
      </div>
    </article>
  );
}
```

Critical constraints — do not deviate:
- Root element is `<article>` not `<div>` — semantic HTML5, cards are self-contained content units (REQ-061)
- `flex-1` on the description `<p>` — pushes links to the bottom so all cards in a row have aligned link rows
- Tag `<li>` styling: `font-mono text-xs px-2 py-1 border border-black/30 tracking-wide` — no background fill, no color (REQ-063, REQ-064)
- Both `<a>` tags carry `target="_blank" rel="noopener noreferrer"` (REQ-033, REQ-053)
- Live link rendered with `{liveUrl && <a ...>}` — renders nothing when liveUrl is undefined or falsy
- `hover:text-black/60` is the only hover effect — no scale, no shadow, no animation
